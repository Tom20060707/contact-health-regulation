2. [这篇论文到底想讲什么](./01-这篇论文到底想讲什么.md)：看现在准备怎么写论文。
3. [这些名字怎么来的](./02-这些名字怎么来的.md)：看 TC-PPO、FTC、CTS、CSC 的来龙去脉。
4. [公式和算法备忘](./03-公式和算法备忘.md)：查公式、变量、算法模块。
5. [术语和缩写](./04-术语和缩写.md)：查容易混的词。
6. [接下来改什么](./05-接下来改什么.md)：看后续论文工作。
